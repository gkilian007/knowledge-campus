# agents-wing-room-01.tmx template

## Purpose
Use Cases room.

## Required layers
- `ground`
- `walls`
- `furniture`
- `collision`
- `hotspots`
- `transitions`

## Minimum composition
- desks or stations representing 3 use cases
- one main hotspot object: `use-cases-main-hotspot`
- one optional secondary hotspot for before/after example
- one exit transition to the Agents room

## Visual direction
Use `office-main` as the dominant visual source.
Optional support from `interiors-support`.
