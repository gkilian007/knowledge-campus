# agents-wing-room-03.tmx template

## Purpose
Builder room.

## Required layers
- `ground`
- `walls`
- `furniture`
- `collision`
- `hotspots`
- `transitions`

## Minimum composition
- one focal build station / kiosk
- one main hotspot object: `builder-main-hotspot`
- optional decorative UI accents from `ui-main`
- one transition backward to Agents room

## Visual direction
Use `office-main` with selective `ui-main` accents to make this room feel like the final practical output station.
