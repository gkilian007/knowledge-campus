# agents-wing-room-02.tmx template

## Purpose
Agents room.

## Required layers
- `ground`
- `walls`
- `furniture`
- `collision`
- `hotspots`
- `transitions`

## Minimum composition
- at least one terminal / workstation zone
- comparison wall or board area
- one main hotspot object: `agents-main-hotspot`
- one transition forward to Builder room
- one transition backward to Use Cases room

## Visual direction
Use `office-main` as dominant source.
Prioritize desks, terminals, monitors, and focused workstations.
